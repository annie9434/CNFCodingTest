# 공원 산책
## 문제풀이
```java
	public int[] solution(String[] park, String[] routes) {
		
		int[] answer = new int[2];
		
        int x = 0; // x좌표 (열)
        int y = 0; // y좌표 (행)
        
        char[][] pArr = new char[park.length][park[0].length()]; // 공원 2차원 배열
        
        // 2차원 배열로 변환 및 강아지 초기 위치 설정
        for(int i = 0; i < park.length; i++){
        	pArr[i] = park[i].toCharArray();
            
            if(park[i].contains("S")){
                y = i;
                x = park[i].indexOf("S");
            }
        }
    
        // 산책 시작
        for(String route : routes){
        	String[] command = route.split(" ");
            String direction = command[0]; //E
            int distance = Integer.parseInt(command[1]); //2
            
            int tmpX = x; 
            int tmpY = y;
            
            for(int i = 1; i <= distance; i++){ 
            	switch (direction) {
	                case "E":
	                    tmpX++; // 동쪽으로 이동 (앞)
	                    break;
	                case "W":
	                    tmpX--; // 서쪽으로 이동 (뒤)
	                    break;
	                case "S":
	                    tmpY++; // 남쪽으로 이동 (아래)
	                    break;
	                case "N":
	                    tmpY--; // 북쪽으로 이동 (위)
	                    break;
                }
            	
            	// 공원을 벗어나지 않고 장애물을 만나지 않았을때 
                if(tmpX >=0 && tmpY >=0 && tmpY < pArr.length && tmpX < pArr[0].length){
                    if(pArr[tmpY][tmpX] == 'X'){
                        break;
                    }
                    // 최종 위치일 때 좌표 저장
                    if(i == distance){
                        x = tmpX;
                        y = tmpY;
                    }
                }
            }
        }       
        
        answer[0] = y;
        answer[1] = x;
        return answer;
    }
```
