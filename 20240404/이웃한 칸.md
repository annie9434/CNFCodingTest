## 문제풀이
```java
    public int solution(String[][] board, int h, int w) {
        int answer = 0;
        
        // h와 w의 변화량을 저장할 정수 리스트
        int[] dh = {0, 1, -1, 0};
        int[] dw = {1, 0, 0, -1};
        
        for(int i = 0; i < 4; i++) {
        	int h_check = h + dh[i];
        	int w_check = w + dw[i];
        	if (h_check >= 0 && h_check < board.length && w_check >= 0 && w_check < board.length) {
        		if (board[h][w].equals(board[h_check][w_check])) {
                	answer++;
                }
            }
        }
        
        return answer;
    }
```
