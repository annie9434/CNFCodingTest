```java
// Q. 초침이 시침/분침과 겹칠 때마다 알람이 울리는데 특정 시간동알 알람이 울린 횟수

// 분침과 초침이 겹치는 횟수
// 59번/3600초
// 분침이 한바퀴 도는데 60분(60번) 움직임, 그러나 처음과 마지막 위치가 겹치므로 -1
// 60(초) * 60(분) 


// 시침과 초침이 겹치는 횟수
// 719번/43200초
// 시침이 한바퀴 도는데 12시간(720분) 720번 움직임,  그러나 처음과 마지막 위치가 겹치므로 -1
// 60(초) * 60(분) * 12(시간)

// 12:00:00 의 경우일 때
// 분침과 초침이 겹치는 경우와 시침과 초침 겹치는 경우가 모두 포함되므로 제외시켜줘야한다.
```
