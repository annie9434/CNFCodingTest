## 문제풀이
```java
public int solution(int n) {

    //계산된 결과를 배열에 저장하여 중복 계산을 피한다.
    int[] fibo = new int[n + 1];
    fibo[0] = 0;
    fibo[1] = 1;

    for (int i = 2; i <= n; i++) {
        fibo[i] = (fibo[i - 1] + fibo[i - 2]) % 1234567;
    }

    return fibo[n];
}
```
